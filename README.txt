    Este portfólio está organizado pela ordem dos slides das aulas, ou seja, o número de cada pasta que aqui se encontra, é igual ao número do PPT da aula correspondente.
    Dentro de cada pasta, encontrará o código desenvolvido para cada um dos assignments (exercícios), organizado por número. Para alguns dos assignments achei necessário desenvolver as partes do código necessárias separadamente, para depois as juntar no código final. Assim, dentro de cada pasta poderá encontrar também pastas cujo nome não corresponde ao número do exercício, mas que são necessárias para o código final.
